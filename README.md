# Moving Average Convergence Divergence

## Opis
Projekt analizuje historyczne ceny platyny z wykorzystaniem wskaźnika MACD. Aplikacja oblicza EMA, MACD i linię sygnału, generuje sygnały kupna i sprzedaży oraz symuluje działanie prostej strategii inwestycyjnej.

## Funkcjonalności
- wczytywanie danych z pliku CSV,
- obliczanie wykładniczych średnich kroczących (EMA),
- wyznaczanie wskaźnika MACD i linii sygnału,
- wykrywanie sygnałów kupna i sprzedaży,
- symulacja portfela inwestycyjnego,
- wizualizacja wyników przy użyciu Matplotlib.

## Technologie
- Python
- Matplotlib

## Struktura projektu
- `data.csv` – historyczne ceny platyny,
- `main.py` – implementacja algorytmów i symulacji.
